# Personal Expense Lister - Java

A java program that enables users to log expenses, view all recorded expenses, and compute total spending.

## Features
- Add Expense (Date | Time | Category | Amount)
- View All Expenses
- View Total Spending
- Records Text File

## Skills
- Programming fundamentals (Java)
- Menu program design
- Loops and ArrayList usage
- I/O file
- String split and parsing
